GIT

Git是目前世界上最先进的分布式版本控制系统

示例

每个人都有一个独立的完整版本库，不依赖于网络，git拥有更为健壮的分支系统，你可以快速地与其他版本库进行合并操作，包括远程服务器上的版本库。git是自由、灵活的代名词（正式因为自由和灵活，有时候git让人看上去特别的复杂，尤其是分支众多的时候）。
